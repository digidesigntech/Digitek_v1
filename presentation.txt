1. WHAT IS THE WEBSITE FOR?
----------------------------------------------------------------
Purpose: Corporate brand + marketing + lead-generation site for
Baptist Digitek Private Limited, a Chennai-based website
development and IT solutions company.

The site is designed to:
  - Establish brand authority in the Chennai / Tamil Nadu market.
  - Showcase services (web, software, mobile apps, hosting,
    branding, digital marketing, corporate training).
  - Generate qualified leads via contact form, WhatsApp, and
    direct phone CTAs.
  - Highlight portfolio work to build trust with prospects.
  - Serve as a discovery channel for the "Digi Design" sub-brand
    (with its own portfolio, packages, and client area).

4. HOW DOES IT LOOK ON MOBILE?
----------------------------------------------------------------
Fully responsive, mobile-first design.

  - Tailwind responsive breakpoints (sm / md / lg / xl) used
    across every component.
  - Navbar collapses to a hamburger menu on small screens.
  - Hero, card stacks, and grids reflow to single-column layouts
    on phones.
  - Touch-friendly tap targets (buttons sized >= 44px).
  - Horizontal overflow guarded globally
    (overflow-x-hidden on body and main).
  - 3D / Spline scenes degrade gracefully on lower-power
    devices; mascot loader provides a smooth perceived-load
    experience.
  - Tested across iOS Safari, Android Chrome, and desktop
    Chrome / Edge / Firefox.


5. HOW FAST DOES IT LOAD?
----------------------------------------------------------------
Performance optimisations in place:

  - Next.js App Router with React Server Components — only
    interactive components ship JavaScript to the client.
  - Automatic code splitting per route.
  - next/font for self-hosted Inter (no external font request,
    zero layout shift).
  - next/image for responsive, lazy-loaded, AVIF/WebP images.
  - Preconnect + DNS-prefetch hints to the Spline CDN to shave
    ~200-500ms off heavy 3D scene loads.
  - Heavy 3D / animation libraries are dynamically imported
    only where needed.
  - Tailwind's JIT compiler keeps CSS small (purges unused
    classes at build time).
  - Static pre-rendering for marketing pages → instant
    first paint from CDN edge.

Typical target: Lighthouse Performance 85+ on mobile,
90+ on desktop. First Contentful Paint under 1.5s on 4G.

7. HOW IS IT KEPT SECURE?
----------------------------------------------------------------
  - HTTPS / SSL: automatic SSL certificate via the hosting
    provider (Vercel auto-renews Let's Encrypt certs).
  - HTTP -> HTTPS redirect enforced at the platform level.
  - No raw database on the marketing site — reduced attack
    surface.
  - All third-party scripts loaded from trusted CDNs over HTTPS.
  - Form submissions can be protected with rate-limiting and
    bot detection (Vercel BotID / hCaptcha if needed).
  - Dependencies kept current; npm audit / Dependabot alerts
    monitored.
  - Source code in private Git repository with branch
    protection and code review.
  - Backups: Git history is the source of truth; the host
    keeps every deployment as an immutable snapshot — instant
    rollback to any previous version.
  - Environment secrets stored in the platform's encrypted
    env-var store, never committed to source.


8. HOW DO WE UPDATE CONTENT?
----------------------------------------------------------------
Current model (developer-managed):
  - Content lives in the codebase (React / TSX components).
  - Updates are made via Git commits and deployed automatically
    on push (continuous deployment via the hosting provider).
  - Baptist Digitek's development team handles edits, with a
    clear change-request workflow for the client.

Optional future model (client self-service):
  - Plug a headless CMS into the existing Next.js site:
      - Sanity, Contentful, Strapi, or Payload CMS.
  - Marketing team edits text / images / blog posts in a
    web dashboard; the site rebuilds automatically.
  - Keeps the performance and security benefits of a static /
    server-rendered site while giving non-developers an easy
    update path.
